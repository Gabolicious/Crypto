Cryptography


This website combines all 8 ciphers taught in Ms. Neely's Cryptography class into a simple to use encrypter and decrypter.

Each page has a short description about what the cipher is and how it works. If it has an equation it'll be listed as well.
You can type your message into the left box, select encrypt or decyprt then input your variables, and then copy the encrypted or decrypted message from the right box.
