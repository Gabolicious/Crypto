Affine Cipher

A monoalphabetic cipher that uses both multiplication and addition (adjusted to Ms. Neely way). A = 1
Encrypt = [(character code + b) * a] mod 26
Decrypt = [(character code * modular multiplicative inverse of a) - b] mod 26
